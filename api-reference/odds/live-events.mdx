<Warning>
  **Requires Authentication**: This endpoint requires an API key and costs **1 credit** per request.
</Warning>

Returns all events that are currently in progress across all sports. Updated every **10 seconds** with real-time scores, match statistics, and live odds.

Perfect for building live scores dashboards or real-time betting applications.

## Authorization

<ParamField header="Authorization" type="string" required default="Bearer " placeholder="Bearer sk_live_...">
  Your API key for authentication
</ParamField>


## Query Parameters

<ParamField query="limit" type="string" default="20">
  Maximum number of live events to return

  Max: `100`
</ParamField>

<ParamField query="prioritize" type="string">
  Smart sorting - prioritize popular leagues first

  Values: `true` or `false`

  When enabled, events from major leagues (Premier League, NBA, Champions League, etc.) appear first.
</ParamField>

## Response

Returns an object with a `data` array containing live events with real-time information.

<ResponseField name="data" type="array">
  Array of live event objects (same structure as [Get Event Details](/api-reference/odds/event-details) but filtered to live events only)
</ResponseField>

### Live Event Includes

Each live event includes:
- Current scores
- Match phase (1st half, 2nd half, quarter, etc.)
- Match statistics (corners, cards, fouls, etc.)
- Live odds for all markets
- Time elapsed

## Example Request

```bash
curl -H "Authorization: Bearer sk_live_..." \
  "https://api.mrdoge.co/odds/live-events?prioritize=true&limit=10"
```
