title: "List Events"
api: "GET https://api.mrdoge.co/odds/events"
auth: "bearer"
description: "Get sporting events with filtering by competition, region, date, and status"
---

<Warning>
  **Requires Authentication**: This endpoint requires an API key and costs **1 credit** per request.
</Warning>

Returns a list of sporting events with basic information. Use filters to find specific events by competition, region, date, or status.

This is the main endpoint for browsing events. For full event details including markets and odds, use the [Get Event Details](/api-reference/odds/event-details) endpoint.

<Note>
  **Pagination**: This endpoint returns a maximum of 100 events per request. To retrieve more events, use date ranges (`startDate`/`endDate`) or filter by specific competitions/regions and make multiple requests.
</Note>

## Authorization

<ParamField header="Authorization" type="string" required default="Bearer " placeholder="Bearer sk_live_...">
  Your API key for authentication
</ParamField>

## Query Parameters

<ParamField query="competitionId" type="string">
  Filter by competition ID

  Example: `1` for Premier League
</ParamField>

<ParamField query="regionId" type="string">
  Filter by region ID

  Example: `1` for England
</ParamField>

<ParamField query="status" type="string">
  Filter by event status

  Options:
  - `upcoming` - Events that haven't started
  - `live` - Currently ongoing events
  - `completed` - Finished events
</ParamField>

<ParamField query="date" type="string">
  Filter events on specific date (YYYY-MM-DD)

  Example: `2025-11-08`
</ParamField>

<ParamField query="startDate" type="string">
  Filter events starting from this date
</ParamField>

<ParamField query="endDate" type="string">
  Filter events ending on this date
</ParamField>

<ParamField query="limit" type="string" default="100">
  Maximum number of events to return

  Default: `100`, Max: `100`

  Results are paginated - use filters to narrow down results or make multiple requests with different date ranges.
</ParamField>

<ParamField query="includeCompleted" type="string" default="false">
  Include completed events

  Values: `true` or `false`
</ParamField>

<ParamField query="timezone" type="string">
  Timezone for date filtering

  Example: `America/New_York`, `Europe/London`
</ParamField>

## Example Request

<CodeGroup>
```bash cURL
curl -H "Authorization: Bearer sk_live_..." \
  "https://api.mrdoge.co/odds/events?competitionId=1&date=2025-11-08&limit=10"
```

```javascript JavaScript
const response = await fetch('https://api.mrdoge.co/odds/events?competitionId=1&date=2025-11-08&limit=10', {
  headers: {
    'Authorization': 'Bearer sk_live_...'
  }
});

const data = await response.json();
console.log(data);
```

```python Python
import requests

headers = {
    'Authorization': 'Bearer sk_live_...'
}

response = requests.get(
    'https://api.mrdoge.co/odds/events',
    headers=headers,
    params={
        'competitionId': '1',
        'date': '2025-11-08',
        'limit': '10'
    }
)

data = response.json()
print(data)
```
</CodeGroup>

## Pagination Strategy

Since the endpoint returns a maximum of 100 events per request, use these strategies to retrieve more data:

### By Date Range

```bash
# Week 1: Nov 1-7
curl -H "Authorization: Bearer sk_live_..." \
  "https://api.mrdoge.co/odds/events?startDate=2025-11-01&endDate=2025-11-07&limit=100"

# Week 2: Nov 8-14
curl -H "Authorization: Bearer sk_live_..." \
  "https://api.mrdoge.co/odds/events?startDate=2025-11-08&endDate=2025-11-14&limit=100"
```

### By Competition

```bash
# Get Premier League events
curl -H "Authorization: Bearer sk_live_..." \
  "https://api.mrdoge.co/odds/events?competitionId=847&limit=100"

# Get La Liga events
curl -H "Authorization: Bearer sk_live_..." \
  "https://api.mrdoge.co/odds/events?competitionId=29&limit=100"
```

### By Region

```bash
# Get English events
curl -H "Authorization: Bearer sk_live_..." \
  "https://api.mrdoge.co/odds/events?regionId=6&limit=100"

# Get Spanish events
curl -H "Authorization: Bearer sk_live_..." \
  "https://api.mrdoge.co/odds/events?regionId=5&limit=100"
```
